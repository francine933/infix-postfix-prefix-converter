# Infix to Postfix and Prefix Converter

Java program that converts infix expressions to postfix and prefix notation using a Stack data structure.

## How to Run
1. Compile: `javac InfixConverter.java`
2. Run: `java InfixConverter`

## Example Output
Input: `A+B*C` 
Postfix: `ABC*+` 
Prefix: `+A*BC`

## Concepts Used
- Stack data structure (LIFO)
- Operator precedence handling
- String manipulation
